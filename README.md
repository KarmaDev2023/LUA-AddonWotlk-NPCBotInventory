# LUA-AddonWotlk-NPCBotInventory

Module complémentaire World of Warcraft 3.3.5 à utiliser avec les NPCBots qui affiche les objets équipés du bot dans une fenêtre.

Cet addon fonctionne en analysant les éléments liés dans le chat par des robots et en les analysant dans une nouvelle fenêtre.

Comme d'habitude avec les addons, décompressez dans Interface/Addons

Utilisation : Connectez-vous et demandez à un bot d'afficher les objets actuellement équipés (Gérer l'équipement - Montrez-moi votre inventaire)
